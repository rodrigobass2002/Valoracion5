Botones en Bootstrap 5

Bootstrap proporciona clases predefinidas para crear botones con estilos consistentes. La clase base btn es obligatoria, seguida de una clase de variante de color.

01

Botones Básicos (Solid)

Variantes de color con fondo sólido

HTML 
<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-light">Light</button>
<button class="btn btn-dark">Dark</button>
<button class="btn btn-link">Link</button>

¿Cómo funciona?

  • btn → Clase base obligatoria (padding, border-radius, cursor)
  • btn-{color} → Define el color de fondo y texto
  • Los colores están definidos en las variables CSS de Bootstrap
02

Botones Outline (Solo borde)

Variantes con fondo transparente y borde de color

HTML 
<button class="btn btn-outline-primary">Primary</button>
<button class="btn btn-outline-secondary">Secondary</button>
<button class="btn btn-outline-success">Success</button>
<button class="btn btn-outline-danger">Danger</button>
<button class="btn btn-outline-warning">Warning</button>
<button class="btn btn-outline-info">Info</button>
<button class="btn btn-outline-dark">Dark</button>
Tip: Los botones outline cambian a fondo sólido al hacer hover. Perfectos para acciones secundarias.
03

Tamaños de Botones

Variantes de tamaño para diferentes contextos

HTML 
<button class="btn btn-primary btn-lg">Botón Grande</button>
<button class="btn btn-primary">Botón Normal</button>
<button class="btn btn-primary btn-sm">Botón Pequeño</button>
04

Estados de Botones

Activo, deshabilitado y con loading

HTML 
<!-- Estado activo -->
<button class="btn btn-primary active">Activo</button>

<!-- Estado deshabilitado -->
<button class="btn btn-primary" disabled>Deshabilitado</button>

<!-- Estado loading con spinner -->
<button class="btn btn-primary" disabled>
    <span class="spinner-border spinner-border-sm"></span>
    Cargando...
</button>
05

Botones con Iconos

Combinando texto e iconos de Bootstrap Icons

HTML 
<!-- Icono a la izquierda -->
<button class="btn btn-primary">
    <i class="bi bi-download me-2"></i>Descargar
</button>

<!-- Icono a la derecha -->
<button class="btn btn-outline-primary">
    Siguiente<i class="bi bi-arrow-right ms-2"></i>
</button>

<!-- Solo icono -->
<button class="btn btn-dark">
    <i class="bi bi-github"></i>
</button>

Espaciado con utilidades

  • me-2 → margin-end (derecha) de 0.5rem
  • ms-2 → margin-start (izquierda) de 0.5rem
06

Grupos de Botones

Agrupando botones relacionados

HTML 
<div class="btn-group" role="group">
    <button class="btn btn-primary">Izquierda</button>
    <button class="btn btn-primary">Centro</button>
    <button class="btn btn-primary">Derecha</button>
</div>

<!-- Grupo con iconos -->
<div class="btn-group" role="group">
    <button class="btn btn-outline-secondary"><i class="bi bi-text-left"></i></button>
    <button class="btn btn-outline-secondary"><i class="bi bi-text-center"></i></button>
    <button class="btn btn-outline-secondary"><i class="bi bi-text-right"></i></button>
</div>

Botones en Tailwind CSS

Tailwind no tiene componentes predefinidos. Construyes cada botón combinando clases de utilidad. Esto te da control total sobre el diseño.

01

Botones Básicos (Solid)

Construyendo botones desde cero con utilidades

HTML 
<button class="bg-blue-500 hover:bg-blue-600 text-white 
               font-medium py-2 px-4 rounded transition">
    Primary
</button>

<button class="bg-green-500 hover:bg-green-600 text-white 
               font-medium py-2 px-4 rounded transition">
    Success
</button>

<button class="bg-red-500 hover:bg-red-600 text-white 
               font-medium py-2 px-4 rounded transition">
    Danger
</button>

Anatomía del botón Tailwind

bg-blue-500 Color de fondo
hover:bg-blue-600 Color en hover
text-white Color del texto
font-medium Peso de fuente
py-2 px-4 Padding Y y X
rounded Border radius
transition Animación suave
02

Botones Outline

Fondo transparente con borde

HTML 
<button class="bg-transparent hover:bg-blue-500 
               text-blue-500 hover:text-white 
               font-medium py-2 px-4 
               border border-blue-500 rounded transition">
    Primary
</button>
Patrón: bg-transparent + border + hover:bg-{color} + hover:text-white
03

Tamaños de Botones

Ajustando padding y tamaño de texto

HTML 
<!-- Grande -->
<button class="... py-3 px-6 text-lg rounded-lg">Grande</button>

<!-- Normal -->
<button class="... py-2 px-4 rounded">Normal</button>

<!-- Pequeño -->
<button class="... py-1 px-3 text-sm rounded">Pequeño</button>

<!-- Extra pequeño -->
<button class="... py-1 px-2 text-xs rounded">Extra pequeño</button>
04

Variantes de Bordes

Diferentes estilos de border-radius

HTML 
<button class="... rounded-none">Sin bordes</button>
<button class="... rounded-sm">Pequeño</button>
<button class="... rounded">Normal</button>
<button class="... rounded-lg">Grande</button>
<button class="... rounded-xl">Extra grande</button>
<button class="... rounded-full">Pill</button>
05

Estados y Efectos

Hover, focus, active y disabled

HTML 
<!-- Con focus ring -->
<button class="... focus:ring-4 focus:ring-blue-300">
    Con focus ring
</button>

<!-- Con efecto scale -->
<button class="... hover:scale-105 active:scale-95 transform">
    Con scale
</button>

<!-- Con shadow en hover -->
<button class="... hover:shadow-lg hover:shadow-blue-500/50">
    Con shadow
</button>

<!-- Disabled -->
<button class="... cursor-not-allowed opacity-60" disabled>
    Disabled
</button>
06

Botones con Iconos

Usando flexbox para alinear iconos

HTML 
<!-- Con inline-flex y gap -->
<button class="... inline-flex items-center gap-2">
    <i class="bi bi-download"></i>
    <span>Descargar</span>
</button>

<!-- Botón circular solo icono -->
<button class="... p-2 rounded-full">
    <i class="bi bi-github text-xl"></i>
</button>
Clave: inline-flex items-center gap-2 alinea icono y texto perfectamente
07

Botones con Gradientes

Efectos de gradiente únicos

HTML 
<button class="bg-gradient-to-r from-blue-500 to-purple-600 
               hover:from-blue-600 hover:to-purple-700
               text-white font-medium py-2 px-6 rounded-lg">
    Blue → Purple
</button>
